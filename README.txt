Color Classifier
Created by Konrad Ceglarski
2019-2020

ENG-POL

It's a simple machine learning (ML) based color classifier made with sklearn,
that has some simple pygame GUI. It is my first program using ML technology.
I've also created a small command line interface (CLI) for it, that offers 5 commands.
By that user can change the language of the main program or give it a color to name.
There's even a little coin flipping game integrated that's ranomly choosing a 0 or a 1.

Jest to prosty klasyfikator kolorów oparty na nauczaniu maszynowym (ML),
stworzony z biblioteką sklearn oraz pygame nadającą interfejs graficzny (GUI).
Jest to mój pierwszy program wykorzystujący technologię ML.
Stworzyłem także prosty i mały interfejs wiersza poleceń (CLI), oferujący 5 komend.
Poprzez to CLI użytkownik może zmienić język programu głównego lub dać mu kolor do nazwania.
Wbudowana jest takża mała gierka, a mianowicie rzut monetą, gdzie losowane jest 0 lub 1.
